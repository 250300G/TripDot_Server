2. Les Composants (Dossier src/components/)

Ce sont les morceaux de code réutilisables qui composent tes pages.

    Navbar.jsx : La barre de navigation avec le logo TripDot.

    Footer.jsx : Le bas de page (celui qui bloquait ton code).

    TripCard.jsx : La petite carte affichant un résumé du voyage dans la liste.

    ActivityCard.jsx : La carte détaillée pour chaque activité (avec prix et heure).

    AddActivity.jsx : Le formulaire pour ajouter une activité à un jour précis.

    WeatherWidget.jsx : (Optionnel) Le petit module météo pour la page détails.



Fichiers de configuration (Racine de src/)

    App.jsx : Là où tu définis tes <Routes> et tes imports globaux.

    main.jsx : Le point d'entrée qui contient BrowserRouter et l'import de Bootstrap.

    index.css : Tes styles personnalisés (couleurs TripDot, polices).
